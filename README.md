# Task Dependency Management System

A full-stack application where tasks can depend on other tasks.
The system prevents circular dependencies, auto-updates task status,
and visualizes task relationships using a graph.

## Tech Stack
- Frontend: React, Tailwind CSS, Canvas/SVG
- Backend: Django, Django REST Framework
- Database: MySQL

## Features
- Create and manage tasks
- Add task dependencies
- Detect circular dependencies (DFS)
- Auto-update task status
- Visual dependency graph
